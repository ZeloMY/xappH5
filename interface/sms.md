# 验证码
 相关信息 | 内容
 ------ | ------
 访问类型 | POST
 访问地址 | /user/sms

###请求参数

 参数名称 | 是否必填 | 类型 | 说明 | 示例数据
 ------ | ------ | ------ | ------ | ------ | ------
 mobile | 是 | 字符串 | 用户手机号 | 
 ip | 是 | 字符串 | 用户ip地址 | 

###请求示例
```javascript
{
   "mobile": "用户手机号",
   "ip": "ip地址"
}
```

###返回结果示例

```javascript
  {
     "code": 1,
     "message": "验证码发送成功",
     "result_data":null
  }
```
