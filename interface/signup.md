# 用户注册
 相关信息 | 内容
 ------ | ------
 访问类型 | POST
 访问地址 | /user/signup

###请求参数

 参数名称 | 是否必填 | 类型 | 说明 | 示例数据
 ------ | ------ | ------ | ------ | ------ | ------
 mobile | 是 | 字符串 | 手机号 | 
 sms | 是 | 字符串 | 验证码 | 
 pwd | 是 | 字符串 | 密码 | 

###请求示例
```javascript
{
   "mobile": "13811111111",
   "sms": "验证码",
   "pwd": "加密后的密码字符串"

}
```

###返回结果示例

```javascript
  {
     "code": 1,
     "message": "注册成功",
     "result_data":null
  }
```
